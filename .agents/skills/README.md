# Unbox Project Skills

This project exposes reusable agent skills in `.agents/skills/`:

- `unbox-prd-alignment` - PRD and cross-doc requirement alignment.
- `unbox-ux-spec` - UX specification updates with REQ/NFR mapping.
- `unbox-brand-voice` - brand-consistent UX copy and product messaging.
- `unbox-engineering-spec` - engineering architecture, schema, and SLO updates.
- `unbox-backlog-execution` - sprint planning with owner/estimate/dependency/blocker.
- `unbox-test-release-ops` - test matrix, release gates, evidence, and risk tracking.
- `unbox-threat-modeling` - structured threat modeling and mitigation planning.
- `unbox-redteam-security` - adversarial red-team testing and security validation.
- `unbox-tracker-imports` - CSV exports for Linear/Jira imports.
- `openrouter-typescript-sdk` - OpenRouter TypeScript SDK integration and usage patterns.
- `promptfoo-evals` - prompt evaluation workflows and quality benchmarking.
- `okx-onchainos` - OKX Onchain OS and X Layer workflows.

Use skill names directly when prompting the agent for focused updates.
